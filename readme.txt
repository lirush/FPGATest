## 1. Сумматор с переносом	(AdderWithCarry)

- Реализован модуль 4-битного сумматора с учетом переноса.
- Протестирован с помощью тестбенча.

## 2. Регистр с асинхронным сбросом и синхронной загрузкой (AsyncResetSyncLoadRegister)

- Создан модуль 4-битного регистра с асинхронным сбросом и синхронной загрузкой данных при сигнале `load`.
- Протестирован регистр с использованием тестбенча.

## 3. Сдвиговый регистр (UniversalShiftRegister)

- Реализован 4-битный универсальный сдвиговый регистр с поддержкой сдвига влево и вправо.
- Протестировано поведение регистра через тестбенч.

## 4. Простой обмен данными по 4-битной шине (FourBitDataBus)

- Создан модуль для передачи 4-битных данных с проверкой готовности через сигнал `valid_in` и `valid_out`.
- Реализован и протестирован модуль с помощью тестбенча.

## 5. Декодер 2-to-4 (TwoToFourDecoder)

- Реализован декодер 2-to-4, который преобразует 2-битный вход в 4-битный выход.
- Проведен тест с мониторингом выходов в тестбенче.

## 6. Многорежимный счетчик (MultiModeCounter)

- Создан модуль многорежимного 4-битного счетчика, который поддерживает счет вверх, вниз, сброс и загрузку значения.
- Протестирована работа счетчика с различными режимами через тестбенч.
